##### 命令
  
  ###### command[-option] [parameter]

   -  command: 命令本身
   -  -option: 命令的一些选项，可以通过选项控制命令的行为细节  非必填
   -  parameter： 命令的参数，多数用于命令的指向目标等

   ```javascript

     ls -l /home/usr
     // 以列表形式，显示/home/usr目录内的内容

     cp -r test1 test2

     // cp本身是命令。 -r是选项，test1 test2是参数，复制文件夹test1成为test2

   ```
